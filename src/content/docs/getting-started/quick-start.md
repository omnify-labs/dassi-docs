This guide walks you through using dassi for the first time.

## Prerequisites

- dassi [installed](/getting-started/installation/) in Chrome
- Signed in with your Google account

## Open the sidebar

Click the **dassi icon** in your Chrome toolbar, or press `Alt+Shift+D` to open the sidebar. You'll see a chat interface with suggested actions for the current page.

## Try your first task

Navigate to any webpage and try one of these:

**Summarize a page:**
```
Summarize this page in 3 bullet points
```

**Draft a reply to an email:**
```
Read this email and draft a professional reply
```

**Fill out a form:**
```
Fill out this form with my name John Smith, email john@example.com
```

**Research across tabs:**
```
Find the pricing for this product on Amazon
and compare with eBay
```

dassi will read the current page, plan its approach, and take action. You'll see each step in the sidebar as it works.

## What you'll see

While dassi works, the sidebar shows:

- **Activity steps** — each action dassi takes (navigating, reading, clicking) appears as a step you can expand to see details
- **A visual indicator** — the active tab shows a glowing border when dassi is controlling it
- **The final response** — dassi's answer or confirmation of completed work

You can click **Stop** at any time to interrupt dassi.
