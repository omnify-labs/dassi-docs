This guide walks you through using dassi for the first time.

## Prerequisites

- dassi [installed](/getting-started/installation/) in Chrome
- An AI provider [connected](/guides/connect-ai-provider/)

## Open the sidebar

Click the **dassi icon** in your Chrome toolbar to open the sidebar. You'll see a chat interface where you can talk to your AI agent.

## Try your first task

Navigate to any webpage and try one of these prompts:

**Summarize a page:**
```
Summarize this page for me
```

**Draft a reply:**
```
Draft a reply to this email
```

**Fill a form:**
```
Fill out this form with my information
```

dassi will read the current page, understand the context, and take action.
