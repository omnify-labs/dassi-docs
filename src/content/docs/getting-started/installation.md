dassi is available as a Chrome extension. Installation takes less than a minute.

## Requirements

- **Google Chrome** (version 120 or later)
- An **AI provider API key** (e.g., from Anthropic, OpenAI, or Google)

## Install from Chrome Web Store

1. Visit the [dassi Chrome Web Store page](https://chromewebstore.google.com/detail/dassi-ai-browser-agent/bjcngahpcjeililljmfegmlanlpgibdi)
2. Click **Add to Chrome**
3. Confirm the installation when prompted

The dassi icon will appear in your Chrome toolbar.

## Pin the extension

For quick access, pin dassi to your toolbar:

1. Click the **puzzle piece icon** (Extensions) in the Chrome toolbar
2. Find **dassi** in the list
3. Click the **pin icon** next to dassi

## Next steps

Once installed, [set up your AI provider](/guides/connect-ai-provider/) to start using dassi.
