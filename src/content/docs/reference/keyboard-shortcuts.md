---
title: Keyboard Shortcuts
description: All keyboard shortcuts available in dassi.
---

## Sidebar

| Shortcut | Action |
|----------|--------|
| `Alt+Shift+D` | Toggle the dassi sidebar open or closed |

## Chat

| Shortcut | Action |
|----------|--------|
| `Enter` | Send your message |
| `Shift+Enter` | Insert a new line without sending |

## Browser control

| Shortcut | Action |
|----------|--------|
| `Ctrl+Shift+C` | Stop dassi from controlling the browser (works from any tab) |
| `Cmd+Shift+C` (Mac) | Stop dassi from controlling the browser (works from any tab) |
| `Esc` | Stop the running agent (only works when the sidebar is focused and the cursor is not in a text input) |

You can also click the **Stop** button in the banner that appears at the bottom of the page while dassi is actively controlling a tab.
